# Contributing

欢迎参与 task-plan-mode 的改进！🎉

## 如何贡献

### 🐛 报告问题
- 通过 GitHub Issues 提交
- 说明：重现步骤、预期行为、实际行为
- 附上 agent 日志或截图更好

### 💡 建议新功能
- 在 Issues 中描述场景和需求
- 说明为什么这个功能对大家有用

### 🔀 提交代码/文档
1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交修改：`git commit -m "feat: add xxx"`
4. 推送：`git push origin feature/your-feature`
5. 提交 Pull Request

### 📖 改进文档
- 修正错别字、不通顺的表述
- 添加更多使用示例
- 翻译更多语言版本

### 🌐 添加新领域脚手架
- 在 `SKILL.md` 的"领域脚手架"章节添加
- 附上典型使用场景说明

## 开发指南

### 本地测试
```bash
# 将 SKILL.md 放入你的 agent skills 目录
cp SKILL.md ~/.openclaw/skills/task-plan-mode/

# 重启 agent 后即可使用
```

### 提交规范
- feat: 新功能
- fix: 修复
- docs: 文档变更
- refactor: 重构
- style: 格式调整

## 行为准则

- 友善、专业
- 接受不同观点
- 关注内容而非人
