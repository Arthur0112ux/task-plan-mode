# 场景三：个人博客网站

## 用户输入

> "我想搭建一个个人博客"

## Plan Mode 过程

**Round 1 — 定位**
Agent：这个博客主要写什么内容？技术博客、生活记录、还是作品展示？
用户：技术博客，分享 AI 和编程相关的内容

**Round 2 — 技术选型**
Agent：有偏好的技术栈吗？比如用 Notion/WordPress 省事，还是自己写代码？
用户：自己写代码吧，顺便练手

**Round 3 — 功能需求**
Agent：需要哪些功能？文章列表、标签分类、搜索、评论系统？
用户：文章 + 标签分类 + 搜索就行，评论先不要

**Round 4 — 设计风格**
Agent：有没有参考的博客网站？或者想要什么风格？
用户：简洁风格，参考 https://example-blog.com

**→ 信息足够，自动退出 Plan Mode**

```
任务总览：个人技术博客
核心需求：文章管理 + 标签分类 + 搜索
技术栈：自写代码，简洁风格
进入执行阶段。
```
