# 场景一：App 开发

## 用户输入

> "帮我做一个记账 App"

## Plan Mode 过程

**Round 1 — 目标用户**
Agent：这个记账 App 主要给谁用？个人记账、家庭共享、还是小商户管理？
用户：个人记账就行

**Round 2 — 核心功能**
Agent：最重要的功能是哪些？比如手动记账、自动导入银行流水、分类统计、预算提醒？
用户：手动记账 + 分类统计 + 预算提醒

**Round 3 — 平台选择**
Agent：做 iOS、安卓、还是微信小程序？
用户：先做小程序试试

**Round 4 — 数据需求**
Agent：需要导出数据吗？比如 Excel/CSV 导出？
用户：要的，方便年底复盘

**→ 信息足够，自动退出 Plan Mode**

```
任务总览：个人记账微信小程序
核心需求：手动记账 + 分类统计 + 预算提醒 + 数据导出
进入执行阶段。
```
