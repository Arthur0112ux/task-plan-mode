# 示例：深度行业分析报告

## 用户输入

> "帮我写一份关于人形机器人行业的深度分析报告"

## Agent 执行过程

### 阶段 1：进入 Plan Mode

Agent 判定：深度研究报告，涉及资料收集、数据分析、结构化写作 → 复杂任务，自动进入 Plan Mode。

### 阶段 2：需求收集（3 轮对话）

**Round 1**
Agent：这份报告的主要读者是谁？投资机构、公司管理层、还是行业从业者？
用户：给制造业的企业管理层看

**Round 2**
Agent：重点想了解哪些方面？技术路线对比、市场格局、还是竞争分析？
用户：重点看竞争格局和商业化进展

**Round 3**
Agent：字数大概多少？有没有特别需要覆盖的企业？
用户：5000-8000 字，重点写优必选、特斯拉、Figure
→ 信息充分，自动退出

### 阶段 3：退出 Plan Mode

```
✅ 信息收集完毕，退出规划模式。

任务总览：人形机器人行业深度分析（管理层版）
核心需求：竞争格局 + 商业化进展，5000-8000字，覆盖优必选/特斯拉/Figure

进入执行阶段。
```
