# 场景二：深度行业报告

## 用户输入

> "写一份人形机器人行业深度分析报告"

## Plan Mode 过程

**Round 1 — 目标读者**
Agent：这份报告的主要读者是谁？投资机构、公司管理层、还是行业从业者？
用户：给制造业的企业管理层看，他们想了解要不要入局

**Round 2 — 关注重点**
Agent：重点想了解哪些方面？技术路线对比、市场规模、竞争格局、还是商业化进展？
用户：重点看竞争格局和商业化进展

**Round 3 — 范围界定**
Agent：有没有特别需要覆盖的企业？以及时间范围？
用户：写优必选、特斯拉、Figure，近 2 年的动态

**Round 4 — 篇幅**
Agent：字数大概多少？
用户：5000-8000 字

**→ 信息足够，自动退出 Plan Mode**

```
任务总览：人形机器人行业深度分析（管理层版）
核心需求：竞争格局 + 商业化进展，5000-8000字
覆盖企业：优必选、特斯拉、Figure
进入执行阶段。
```
