# 场景四：产品设计方案

## 用户输入

> "帮我做一个社区团购小程序的产品方案"

## Plan Mode 过程

**Round 1 — 目标用户**
Agent：这个社区团购主要服务哪个区域？小区居民、办公楼、还是特定社群？
用户：小区居民，先做一个中高档小区试点

**Round 2 — 核心角色**
Agent：涉及哪些角色？团长、供应商、平台方？
用户：团长和普通居民，供应商先由平台代管

**Round 3 — 核心功能**
Agent：最重要的功能是哪些？开团、下单、支付、物流追踪？
用户：团长开团 + 居民下单 + 在线支付

**Round 4 — 商业模式**
Agent：盈利模式是什么？抽成、会员费、还是广告？
用户：每单抽成 5%

**→ 信息足够，自动退出 Plan Mode**

```
任务总览：社区团购小程序产品方案（中高档小区试点）
角色体系：团长 + 居民
核心功能：开团 + 下单 + 支付
盈利模式：每单抽成 5%
进入执行阶段。
```
