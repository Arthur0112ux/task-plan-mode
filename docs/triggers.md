# task-plan-mode 触发词速查表

Agent 根据以下关键词自动判断是否进入任务规划模式。

## 自动触发词

| 触发场景 | 用户可能说的话 | 触发行为 |
|---------|---------------|---------|
| **项目开发** | 帮我做一个XX、开发一个XX、搭建XX平台 | ✅ 进入 Plan Mode |
| **内容创作** | 写一篇XX文章、出一份XX报告、制作XX方案 | ✅ 进入 Plan Mode |
| **研究分析** | 分析一下XX、研究XX行业、对比XX和XX | ✅ 进入 Plan Mode |
| **复杂任务** | 这个任务比较复杂、步骤比较多、涉及多个环节 | ✅ 进入 Plan Mode |
| **模糊需求** | 帮我做个东西、想搞个产品、有个想法 | ✅ 进入 Plan Mode |

## 手动触发词

| 你说这句 | 行为 |
|---------|------|
| 切任务模式 / Plan Mode | ✅ 强制进入规划模式 |
| 先规划一下 / 先拆解一下 | ✅ 进入规划模式 |
| 帮我拆解这个任务 | ✅ 进入规划模式 |
| 这个比较复杂，你先分析一下 | ✅ 进入规划模式 |

## 不触发的情况

以下简单任务不会触发 Plan Mode，直接执行：
- 帮我查一下XX / 搜索XX
- 翻译一段文字
- 总结这篇文章
- 帮我想个名字 / 起个标题
- 帮我改一下这段话
- 现在几点 / 天气怎么样
